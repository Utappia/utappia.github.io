Linux v3.3.1 and v3.2.14 are released

To all Kernel compilers out there... new releases of Linux are on the wild... All users of the 3.2 and 3.3 kernel series must upgrade.

Numerous fixes and features are now in the new versions of Linux Kernel. You may wonder which version you should pick. Here is a quick decision making algorithm :

  • if you are experienced user and you love to be on the edge then choose 3.3.1
  • if you are experienced user and you want to test new features then choose 3.3.1
  • if you want just an update for your existing 3.2.x series then choose 3.2.14

Stay tuned if you would like precompiled packages of Linux kernel patched with BFS/BFQ and in-house optimizations. Soon there will be new packages for download ;)

Until then if you just want vanilla kernel download it from Kernel.org
